## One Capability

Add a fixture-only Commander decision helper that accepts sanitized evidence from CM-1701 / CM-1700 / CM-1699 / CM-1698-style packets and returns a low-disclosure go/no-go decision.

The helper must support at least these decisions:

- `continue_local_safe`: when the requested next action is docs, fixture, source contract, test, or low-disclosure packet hardening and no runtime-sensitive action is requested.
- `blocked_needs_exact_approval`: when the requested next action includes target-specific runtime inspection, live VCPToolBox call, secret/config/private-state access, raw memory/raw runtime output, durable memory write, provider/API call, public MCP expansion, startup/watchdog/config mutation, readiness/cutover/release/deploy/push, or unbounded budget.
- `needs_plan_adjustment`: when the Master/Stage/Version alignment is missing or ambiguous.
